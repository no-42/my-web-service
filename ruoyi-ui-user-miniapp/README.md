### 用户端小程序

#### 相关库
* 使用 tarojs 来跨平台编译
* 使用 vue3 开发
* 使用 nutui 提供基础界面

#### 使用说明
初次使用安装依赖
```shell
npm i
```
开发
```shell
npm run dev:h5
```
打包
```shell
npm run build:weapp
```

#### 相关文档
* [tarojs](https://taro-docs.jd.com/docs/)
* [nutui](https://nutui.jd.com/#/zh-CN/component/button)
